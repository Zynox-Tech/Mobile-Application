### 2. Vehicle Inventory App

**Repository:** [Vehicle_Inventory_App](https://github.com/Zynox-Tech/Vehicle_Inventory_App)

A modern vehicle parts management application built using **Flutter and Firebase**.

The application is designed to digitize inventory operations, billing, customer management, and delivery tracking for vehicle parts businesses.

Key capabilities include **inventory management, QR-based billing, real-time order tracking, customer management, and live delivery monitoring**.

The system provides a complete digital solution for improving operational efficiency and simplifying daily business workflows.

### 3. TrackSitePro

**Repository:** [TrackSitePro — SNGPL Contractor Project Manager](https://github.com/Zynox-Tech/TrackSitePro-sngpl-contractor-project-manager)

A professional mobile-based project management application designed for managing contractor operations, site activities, project tracking, and field workflows.

TrackSitePro provides a centralized digital solution for improving communication, monitoring project progress, and efficiently managing contractor and field operations.

The application is designed to help teams organize workflows, track site activities, and improve operational visibility.
